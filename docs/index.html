🛰️ ESP32-BlueDriver

One-click web installer — a Bluetooth LE wardriver for the ESP32-S3-DevKitC-1 (N16R8).

Install in 30 seconds

  1. Use Chrome or Edge on a desktop (Web Serial required).
  2. Plug the ESP32-S3 into USB.
  3. Click Install, pick the serial port, confirm.

Your browser doesn't support Web Serial — use Chrome or Edge. Open this page over HTTPS to install.

First flash erases the chip and writes the full firmware.

After flashing

  1. The board creates a Wi-Fi network named BlueDriver (password wardrive).
  2. Connect your phone/laptop to it.
  3. A control panel opens automatically — or browse to http://192.168.4.1/.

ESP32-S3 has a Bluetooth LE radio only (no Bluetooth Classic), so this tool wardrives BLE advertisements. For authorized testing and educational use. See the project README.
